# 帮助信息服务

> 所属章节: 系统服务说明

---

获取服务帮助信息，所有的字段属性。

服务接口名称命名规则：yz.oapi.{Typekey}.help

#### 输入字段

| 标签 | 说明 |
| --- | --- |
|  |  |

#### 返回字段

| 标签 | 描述 |
| --- | --- |
| std_data | 固定标签，返回数据的根标签 |
| execution | 固定标签，用于服务调用的状态 |
|  | 状态码：
            0: 表示服务调用成功
           -1: 表示服务调用失败 -0: 表示服务调用部分成功 批量调用时，成功结果返回在result. success ,失败结果返回在result.error |
| description | 状态码为-1时，表示具体异常信息 |
| sql_code | 系统预留字段 |
| parameter | API服务返回内容 |
| result | 其中包含success和error标签，分别存放成功/失败的数据处理状态。 |
| cdsMaster | 数据集名称，新增时需要做对应 |
| datasetName | 数据集名称 |
| tableName | 对应的数据库表名 |
| fieldList | 字段属性集合 |
| name | 字段编号 |
| jsonName | 节点名称 |
| displayLabel | 字段名称 |
| dataType | 字段类型 |
| size | 字段长度 |
| readOnly | 只读 |
| visible | 显示 |
| required | 必填 |
| fieldKind | 字段性质 |
| ConfirmField | 审核字段 |

#### 示例场景

获取生产入库单所有字段信息

## Request

### http-header

#### 示例

Content-Type: application/json; charset=utf-8
digi-service: {"name":"yz.oapi.wo.stock.in.help"}
digi-user-token: Ht7XKsAioLgYgf0+uL34Ar/uv1w315Edkvw29ExXYyA=(此值通过ERP作业 设置授权人员 TPASC19获取)

### http-body

#### 示例

{}
